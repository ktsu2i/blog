こんにちは、筒井（[@ktsu2i](https://x.com/ktsu2i)）です。

タイトルの通り、自分のブログサイトを作りました。

普段は所属企業のテックブログや Zenn をたまーに書いてたんですが、もうちょっと気楽にアウトプットしたいなと思って作ってみました。

いろんなところに自分の書いた記事が散らばってるので、このサイトに集結させました。会社のテックブログとかも全部ここに集まるので便利。

あと、海外向けに英語でも記事書きたいなーと思って、i18n にも対応させてみました。右上のボタンで切り替えられます。

## 技術スタック

普段は Go ばっかり書いてるんですが、今回は Astro で作って、Cloudflare workers にデプロイする構成にしました。超簡単にできちゃいました、Astro すごい。

久しぶりに個人開発をやってみたら、やっぱり楽しくて良いですね。今回、個人開発用に Claude Code の Pro プランに入ったので、これからいっぱいコード書けます。OSS にコントリビュートとかしたいな。

## これから

個人的なアウトプットとか、カンファレンスのレポートなどなど、そういうのを気ままに投稿しようかなと思ってます。よかったら読んでもらえると嬉しいです。
