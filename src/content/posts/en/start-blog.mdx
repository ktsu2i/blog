I'm Kaito Tsutsui ([@ktsu2i](https://x.com/ktsu2i)), a Go-focused software engineer in Tokyo.

As the title says, I built my own blog site.

I've been writing on my company's tech blog and [Zenn](https://zenn.dev) (a dev blogging platform in Japan), but I wanted a more casual place to share my thoughts, so I decided to build one myself.

My articles were scattered across different platforms, so I brought them all together here. It's nice having everything in one place, including posts from my company's tech blog.

I also wanted to write in English for a global audience, so I added i18n support. You can switch languages with the button in the top right corner.
